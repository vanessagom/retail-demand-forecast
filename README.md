# Walmart Demand Forecast

Proyecto de forecasting para estimar la demanda diaria a nivel tienda-categoría utilizando datos históricos de transacciones, características de tiendas y variables de calendario.

## Objetivo

Desarrollar un modelo que anticipe el número diario de transacciones por tienda y categoría para apoyar decisiones de inventario, personal operativo y planeación promocional.
